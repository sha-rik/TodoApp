# Todo App (Role-Based)

A full-stack Node.js application with:
- JWT authentication
- Role-based access (Admin / Student)
- Todo CRUD functionality
- Protected routes

## Tech Stack
- Node.js
- Express
- MongoDB
- JWT
- Vanilla HTML & JavaScript

## Features
- User signup & login
- Role-based redirection
- Student todo dashboard
- Admin dashboard (in progress)

## Setup
1. Clone repo
2. Install dependencies
3. Add `.env`
4. Run `start_TodoApp`
